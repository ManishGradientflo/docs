## **January 2026**

### **Week 4**

- Linear Integration reached general availability, enabling ByteCourse notifications inside Linear issues.
- SonarCloud and CodeAnt integrations are live for detecting security and code-quality gaps.
- Team Projects Dashboard launched for organization-wide skill gap visibility.
- Fixed an issue with webhook retry logic related to GitHub API rate limits.

### **Week 3**

- Introduced the PR-Driven Micro-Loop: Detect → Recommend → Learn → Verify.
- Released Gap Detection v2, which analyzes coverage, security issues, complexity, and missing tests.
- Improved re-analysis latency to under 5 seconds (previously 15 seconds).

### **Week 2**

- Skill Graphs now include evidence links showing which pull request demonstrated each skill.
- Success Comments are automatically posted when a gap is resolved.
- Increased gap detection accuracy by 40%.

### **Week 1**

- Released the Intelligence Layer Foundation for GitHub, Linear, and SonarCloud.
- Deployed real-time webhook infrastructure for processing pull request events.
- Added a GitHub Comment Bot that posts ByteCourse recommendations directly in pull requests.
