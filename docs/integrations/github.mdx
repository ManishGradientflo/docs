## **Privacy**

We access:

- Pull request metadata
- File diffs
- CI/CD results

We do **not** access:

- Secrets
- Environment variables
- Private issues

Storage policy:\
File diffs are cached for 24 hours and then deleted.

---

## **Troubleshooting**

If comments are not appearing in pull requests:

- Check webhook delivery logs in GitHub Settings.
- Verify that the `pull_requests:write` permission is enabled.
- Confirm that the repository is included in the monitored list.
